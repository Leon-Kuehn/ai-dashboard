What are the context window limits for llama3?
llama3:8b · 14:32
The Llama 3 family supports the following context windows:

• llama3:8b — 8,192 tokens (default), extendable to ~128K with RoPE scaling
• llama3:70b — 8,192 tokens (same architecture)

In practice, Ollama defaults to 2,048 tokens for generation but you can override this via the num_ctx parameter in your Modelfile or API request. Keep in mind that larger contexts increase VRAM usage substantially.
